# Venture Capital & PE Event Scheduler and RSVP Manager Bot

This bot streamlines event management for venture capital and private equity firms. It automates scheduling, RSVP tracking, and attendee communications for industry conferences, pitch days, LP meetings, and portfolio company events. With intelligent calendar integration, it syncs with Google Calendar, Outlook, and Slack to avoid double bookings. The bot sends personalized invitations, follow-ups, and reminders via email and SMS. ## Setup
1. Set `TELEGRAM_BOT_TOKEN` in `.env`
2. `pip install -r requirements.txt`
3. `python main.py`
